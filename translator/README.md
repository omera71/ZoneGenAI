# Translator CLI dengan Multiple Providers

Aplikasi CLI untuk menerjemahkan teks dari file menggunakan multiple provider gratis.

## Fitur
- Support multiple translation providers (Google Translate, MyMemory)
- Input dari file teks
- Output ke file teks dengan metadata
- Deteksi bahasa otomatis
- Progress bar dan statistik
- Preview hasil terjemahan
- Support 18+ bahasa
